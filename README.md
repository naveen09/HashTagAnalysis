#Hashtag based sentiment analysis using Twitter API

![HashTagAnalysis](https://raw.githubusercontent.com/naveen09/HashTagAnalysis/master/hashtag_screen.png)

###How to install

```
npm install
```
```
npm install bower
```
```
npm install grunt
```
```
npm install grunt-cli
```
```
bower install
```
```
grunt build
```
```
grunt serve
```
###Tested
* Nodejs v0.10.29
* grunt-cli v0.1.13
* grunt v0.4.5
* bower 1.4.1
